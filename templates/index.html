Documentation

This tool helps you transcribe audio files (mp4, m4a, mp3, ogg) from meetings. Follow these steps:

  1. Select the transcription language from the dropdown menu.
  2. Upload your audio file using the file input.
  3. Click the "Transcribe" button to start the transcription process.
  4. Wait for the transcription to complete and download the resulting text file.
